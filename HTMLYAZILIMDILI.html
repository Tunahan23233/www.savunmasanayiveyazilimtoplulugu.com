HTML

HTML web site yapımında kullanılan yazılım dilidir eğer HTML'de head etiketinin altına style etiketi koyulursa CSS yazılım diline girer.HTML'de genelde tüm etiketlerin başına < (küçüktür) sonuna > (büyüktür) konulur.

HTML sayfası oluşturmak için

ul ve ol etiketleri

ul ve ol etikeleri HTML'de bir sıralama için kullanılan etiketlerdir.Bu etiketlerin içine li etiketi kullanarak sıralama öğeleri yapılmış olur.Fakat ul etiketi işaretlerle sıralama yaparken ol etiketi type ile sıralama yapar yani: A B C , a b c, 1 2 3, I. II. III. gibi sıralama yapar


  • Çay
  • Kahve
  • Süt
<ul style="list-style-type:square">                
   <li>Çay</li>              
   <li>Kahve </li>                
   <li>Süt</li>                
</ul>

Bazen bazı durumlarda ul içinde ul kullanılır örnek

  • Çay
    • Demli çay
    • Demsiz çay
  • Kahve
  • Süt

button etiketi

ü Button etiketi butondur.Eğer butonu bir bağlantı içinde yazarsanız bir uyarı verecektir button tıklama işlevi görürü

}

Script kodları(head altı kodlar)
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script<
<script>
    $(document).ready(function () { <!--İdnin başına # getirilir-->
        $("#butto").click(function () {
            alert($("#text").html()); <!--Burada idsi text olan şeyin yazısını uyarı olarak verir-->
        })
        $("#button1").click(function () {
            $("#text1").html("<b>TÜRK<b>"); <!--Butona basıldığında text idli şeyi parantez içindeki değer yapar-->
        })
        $("#button2").click(function () {
            var deger = $("#birdeger").val(); <!--buttona basıldığında birdegerden alınan veriyi yazı idli şeye yazdırır-->
            $("#yazı").html(deger);
        })
        $("#krm").click(function () {
            $("#yazılacakdeger").val("Hello World"); <!--Buttona basıldığında krm idli şeyin değerini Hello World yapar-->
        })
    })
</script>
